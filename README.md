# 💰 Expense Tracker Web Application

A Flask-based web application that allows users to manage and analyze expenses efficiently.

## 🚀 Features

- Add and view expenses
- Upload expense data using JSON files
- Group expenses by user
- Automatic category-wise calculations
- Dynamic pie chart visualization (Matplotlib)
- Scrollable and responsive UI using Bootstrap
- User-based expense reports
- Clean and modern dashboard layout

## 🛠️ Technology Stack

- Backend: Python, Flask
- Frontend: HTML, CSS, Bootstrap
- Data Handling: JSON
- Visualization: Matplotlib
- Template Engine: Jinja2

## 📊 Chart Features

- Category-wise expense distribution
- Separate pie charts for each user
- Auto-calculated percentage display

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000
```
